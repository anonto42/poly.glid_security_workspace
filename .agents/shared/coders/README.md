# Shared Coder Roles

Use these as lightweight task lenses. They do not replace
`.agents/shared/agent-startup.md`; load startup first, then the role that matches
the work.

- `rust-coder.md` - PolyGlid Rust crates, WIT, WASM plugins, runtime, and CLI.
- `review-coder.md` - code review and risk checks.
- `release-coder.md` - Docker, GitHub Actions, deployment, release notes.

For implementation, still load the matching rules from
`.agents/shared/rules/README.md`.
