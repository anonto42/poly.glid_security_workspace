# Shared Change Notes

Use this folder for durable cross-assistant notes about repository-wide changes,
conventions, migrations, or decisions.

## Conventions

- Keep each note under 200 lines.
- Use clear filenames, for example `2026-05-04-agent-knowledge-base.md`.
- Record what changed, why it changed, and any follow-up needed.
- Do not duplicate git history or store secrets.
