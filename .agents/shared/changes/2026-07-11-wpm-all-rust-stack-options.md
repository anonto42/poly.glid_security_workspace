# WPM All-Rust Stack Options

Date: 2026-07-11

## What changed

Recorded three frontend/backend choices for the requested Rust-first WPM:
Dioxus full-stack, Leptos with Axum, or a native Slint desktop application.

## Current recommendation

Dioxus is preferred when desktop, web, and future mobile should share the most UI
code. Leptos is preferred if WPM becomes browser-first. Selection remains pending.
