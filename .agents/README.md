# Shared Agent Workspace

Canonical AI agent knowledge lives in `.agents/shared/`.

Use:

- `shared/agent-startup.md` for startup.
- `shared/caveman.md` for compressed continuation/review style.
- `shared/history/YYYY-MM-DD/` for daily shared work memory.
- `shared/memory/workspace-ai-current-state.md` for the verified `.workspace` AI status.
- `shared/plans/project-management-agent-platform.md` for the active project-management and sub-agent plan.
- `shared/rules/` and `shared/coders/` only as needed.
- `shared/legacy/` for archived summaries from merged old agent roots.

Do not duplicate instructions under `.claude/` or `codex/` or `.gemini/` or any other AI agent configuration directories.
