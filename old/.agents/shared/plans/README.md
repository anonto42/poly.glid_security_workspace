# Shared Plans

Use this folder only for durable cross-day plans. Daily task plans should go in
`.agents/shared/history/YYYY-MM-DD/plans.md`.

## Conventions

- Keep each plan under 200 lines.
- Use clear filenames, for example `2026-05-04-onboarding-flow.md`.
- Include status, scope, decisions, open questions, and verification needs.
- Update the existing plan instead of creating competing plans for the same
  task.
- Do not store transient chat logs or secrets.
