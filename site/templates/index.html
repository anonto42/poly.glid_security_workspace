Get started

Inspect and run the bundled MVP component:

$ polyglid plugin inspect plugins/recon-probe.component.wasm
id: polyglid.recon_probe
name: Reconnaissance Probe
version: {{ version }}
requested capabilities:
- dns-resolve
- report-write

$ polyglid plugin run plugins/recon-probe.component.wasm \
    --target localhost --allow dns-resolve --allow report-write

target: localhost
summary: 1 demo observation(s) reported for localhost.
